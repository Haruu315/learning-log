# learning/ — 学ぶために書く場所

## ここに入れるもの

- 講座・スクール教材の写経（winc01、web developer 講座 など）
- チュートリアル
- 「○○入門」をやってる時の練習コード
- 本のサンプルコードを動かしてみる

## ここに入れないもの

- 自分のアイデアを試したい試作（→ `sandbox/`）
- ちゃんと作って公開したいもの（→ `projects/`）

## 推奨の整理方法

講座やテーマでフォルダを分ける：

```
learning/
├── winc01/
├── react-tutorial/
├── python-basics/
└── 2026-05-20_paiza-week1/
```

## 完成を求めない場所

ここはあくまで「学ぶための場所」。コードが汚くてもOK。途中で放置してもOK。
完成度を求めずに、ガンガン書く場所。
